# Data Notes

本项目使用的太阳能电池板图像数据未完整上传至 GitHub。

## Data Source

当前本地目录包含 `Defect/` 与 `NoDefect/` 两类图像样本，以及由实验流程生成的 SVD 图像、批处理输出和分割数据集。数据文件名显示其来源于相机采集快照，具体数据授权和公开范围仍需人工确认。

## Data Format

原始与中间数据主要为 `.jpg`、`.png` 等图像文件。分割训练子任务中使用 `images/train|val` 与 `masks/train|val` 的图像/掩膜配对格式。

## Access

原始数据、批量输出、训练集切分和人工标注掩膜默认仅本地保存，不建议直接公开上传。若需要公开演示，应仅保留少量已脱敏、已确认授权的样例图。

## Reproduction

没有原始数据时，仍可阅读 `README.md`、`docs/` 中的报告材料，并查看 `assets/` 与 `results/` 中的少量代表性结果图。若要复现实验，需要准备同类太阳能电池板缺陷图像，并按 README 中的命令运行预处理、SVD 重建和分割流程。
