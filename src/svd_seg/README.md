# SVD 裂纹分割系统（UNet）\n\n目标：基于 SVD 处理后的图像进行像素级裂纹分割与高亮显示。\n\n## 目录结构\n- svd_seg/\n  - README.md\n  - prepare_split_seg.py\n  - annotate_seg_gui.py\n  - train_unet.py\n  - infer_overlay.py\n  - requirements_seg.txt\n  - dataset/\n    - images/train|val\n    - masks/train|val  # PNG 掩膜，与图像同名\n\n## 安装依赖\n````powershell\nD:/Anaconda/Scripts/conda.exe run -p e:\\solarcellcode_DL\\.conda --no-capture-output pip install -r e:\\solarcellcode_DL\\svd_seg\\requirements_seg.txt\n````\n\n## 数据准备（使用 Trainset 源）\n````powershell\nD:/Anaconda/Scripts/conda.exe run -p e:\\solarcellcode_DL\\.conda --no-capture-output python e:\\solarcellcode_DL\\svd_seg\\prepare_split_seg.py --src e:\\solarcellcode_DL\\Trainset --out e:\\solarcellcode_DL\\svd_seg\\dataset --train 0.8\n````\n- 会复制图片到 `dataset/images/train|val`。掩膜初始为空，待人工标注生成至 `dataset/masks/train|val`。\n\n## 掩膜标注（多边形）\n````powershell\nD:/Anaconda/Scripts/conda.exe run -p e:\\solarcellcode_DL\\.conda --no-capture-output python e:\\solarcellcode_DL\\svd_seg\\annotate_seg_gui.py --root e:\\solarcellcode_DL\\svd_seg\\dataset --subset train\n````\n- 在图上点击多点形成多边形；按 `完成多边形` 将当前多边形加入列表；可添加多个多边形；点击 `保存掩膜` 生成同名 PNG 掩膜（白色=裂纹，黑色=背景）。\n\n## 训练 UNet\n````powershell\nD:/Anaconda/Scripts/conda.exe run -p e:\\solarcellcode_DL\\.conda --no-capture-output python e:\\solarcellcode_DL\\svd_seg\\train_unet.py --data e:\\solarcellcode_DL\\svd_seg\\dataset --epochs 50 --img 512 --batch 8\n````\n- 读取 `images/train` 与 `masks/train` 对应文件训练；保存权重到 `svd_seg/runs/unet/best.pt`。\n\n## 推理与高亮叠加\n````powershell\nD:/Anaconda/Scripts/conda.exe run -p e:\\solarcellcode_DL\\.conda --no-capture-output python e:\\solarcellcode_DL\\svd_seg\\infer_overlay.py --data e:\\solarcellcode_DL\\svd_seg\\dataset --weights e:\\solarcellcode_DL\\svd_seg\\runs\\unet\\best.pt --subset val\n````\n- 在 `dataset/images/val` 上推理，保存叠加高亮到 `svd_seg/overlays/val`。\n\n## 说明\n- 模型默认将灰度复制为 3 通道进行训练；你也可切换为单通道分支。\n- 若希望使用 YOLOv8-seg 替代 UNet，我可以提供 YOLO 格式掩膜保存与训练命令（Ultralytics 已安装）。\n
